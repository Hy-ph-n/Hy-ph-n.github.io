Goblin Adventure

A game about traps and gold

When designing this python program, the objective was to have a game with a simple objective, as well as some obstacles in their way. I chose the theme of a goblin seeking wealth because I thought it was a fun way to present this concept, including bombs and barred spaces that the user cannot enter so that the player actually has a way to lose. If the player runs out of lives, they get a lose screen. If they collect all the coins necessary, they get a win screen. I also added special coins that are worth more for some variety. The interesting thing is that the goblin can wrap around the map as long as they aren't entering a space where a barrier is present. The repository for this project can be found here.
